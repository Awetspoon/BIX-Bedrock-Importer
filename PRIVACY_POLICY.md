# BIX Privacy Policy

Effective date: 29 June 2026

BIX (Bedrock Import eXpress) is an independent Android utility for preparing user-selected Minecraft Bedrock files and handing them to the Minecraft app.

## Data handling

BIX does not collect, sell, upload, or transmit personal information to the developer. The app has no advertising, analytics, account system, or third-party tracking SDK.

When a user explicitly selects a file or dedicated add-on folder, BIX processes that content locally on the device. It may create a temporary `.mcpack`, `.mcaddon`, or `.mcworld` file in the app's private cache. When the user taps **Import with Minecraft**, BIX gives the installed Minecraft app temporary read access to that prepared file. Android automatically removes the receiving activity's access when its activity stack finishes. BIX removes cached import files older than 24 hours when the app next runs, and Android may clear app cache sooner.

BIX does not alter or delete the user's original selected file or folder. Android and Minecraft control their own storage and final import behaviour.

BIX can contact GitHub Releases to check whether a newer BIX version is available. This update check sends a normal HTTPS request to GitHub and does not include selected add-on files, worlds, folders, account details, or analytics identifiers. GitHub may process basic network information, such as IP address and request metadata, under GitHub's own privacy policy.

## Children

BIX does not knowingly collect personal information from children or any other users.

## Third-party application

Minecraft is provided by Mojang Studios and Microsoft. GitHub is used for public BIX release downloads and update checks. Those services handle their own data under their own terms and privacy policies. BIX is independent and is not affiliated with, endorsed by, or sponsored by Mojang Studios, Microsoft, or GitHub.
